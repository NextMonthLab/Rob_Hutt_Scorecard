# Marketing Reality Check Scorecard

A simple, production-ready scorecard to assess marketing clarity, human connection, and execution.

## Getting started

### Install dependencies

```bash
npm install
```

### Run the dev server

```bash
npm run dev
```

To run the API server in development (separate terminal):

```bash
npm run dev:server
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Render Web Service settings

- Build Command: `npm install && npm run build`
- Start Command: `npm start`
